# Getting Started with PRISM

Welcome! Let's get you up and running in 5 minutes.

## What is PRISM?

PRISM is a proven 10-phase methodology for building products from idea to launch.

- **Phase 1-2**: Understand the problem
- **Phase 3-5**: Design & build the solution
- **Phase 6-8**: Harden, deploy, monitor
- **Phase 9-10**: Test & launch

## Quick Start (5 minutes)

### 1. Download the Skill
Save [ETPD-SKILL.md](ETPD-SKILL.md) to your computer.

### 2. Prepare Your Brief
Create a file with:
```markdown
# My Project Brief

Project: [Name]
Problem: [What problem?]
Users: [Who will use it?]
Vision: [End goal?]
Budget: [Time/money]
Success Metrics: [How to measure?]
```

### 3. Share with Claude AI
```
I want to build a product using PRISM ETPD methodology.

Here's my brief:
[YOUR BRIEF]

Here's the skill:
[PASTE ETPD-SKILL.md]

Please work through all 10 phases.
```

### 4. Follow the Phases
Claude will guide you through each phase.

### 5. Launch!
You'll have a production-ready product.
