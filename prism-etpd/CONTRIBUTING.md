# Contributing to PRISM

Thank you for wanting to contribute! 🎉

## How to Contribute

### Report Issues
1. Check [existing issues](https://github.com/prism-etpd/issues)
2. [Open a new issue](https://github.com/prism-etpd/issues/new)

### Suggest Improvements
1. [Start a discussion](https://github.com/prism-etpd/discussions/new)
2. Describe your idea clearly

### Improve Documentation
1. Fork the repo
2. Edit documentation
3. Submit a pull request

### Share Your Project
1. Create a folder in `examples/`
2. Add your case study
3. Submit a pull request

## Process

1. Fork the repo
2. Create a branch (`git checkout -b feature/my-feature`)
3. Make changes
4. Commit (`git commit -m 'feat: description'`)
5. Push (`git push origin feature/my-feature`)
6. Open a Pull Request

## Guidelines

✅ Be respectful  
✅ Provide clear descriptions  
✅ Include examples  
✅ Test your changes  
✅ Update documentation
