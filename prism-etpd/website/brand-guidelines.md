# PRISM Brand Guidelines

## Logo Usage

### Primary Logo
Use the full prism graphic + "PRISM" text

### Icon Only
Use just the prism graphic when space is limited

### Text Only
Use "PRISM" in Montserrat 700 when logo is too small

### Clear Space
Keep at least 1/4 of logo height clear space around logo

## Colors

### Primary Colors
- **Deep Blue:** #1a365d (authority, trust)
- **Rich Purple:** #6B46C1 (innovation, creativity)

### Spectrum (Secondary)
- Red: #FF6B6B
- Orange: #FFA500
- Yellow: #FFD700
- Green: #4CAF50
- Blue: #2196F3

### Neutrals
- Dark Gray: #2D3748
- Medium Gray: #718096
- Light Gray: #E2E8F0
- White: #FFFFFF

## Typography

### Primary Font: Montserrat
- Headlines: Weight 700 (bold)
- Subheadings: Weight 600
- Body: Weight 400
- Small text: Weight 400

### Secondary Font: Inter
- Code snippets
- Captions
- Small text

## Imagery Guidelines

### Do ✅
- Use geometric, minimalist imagery
- Feature code and development
- Show real products & projects
- Use spectrum colors
- Show diverse faces & teams

### Don't ❌
- Use generic stock photos
- Add unnecessary shadows/effects
- Mix multiple design styles
- Use outdated imagery
- Distort the logo

## Website Styling

- Accent Color: Purple #6B46C1
- Hover States: Lighter purple
- Success: Green #4CAF50
- Warning: Orange #FFA500
- Error: Red #FF6B6B
- Info: Blue #2196F3
