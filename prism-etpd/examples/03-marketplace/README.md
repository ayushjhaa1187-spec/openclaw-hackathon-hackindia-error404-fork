# TechMarket: Case Study

A two-sided B2C marketplace platform for digital assets built entirely with PRISM.

## The Project

**Timeline:** 16 sessions  
**Result:** Turn-key marketplace platform  

## Phase Highlights

- **Phase 1-2 (Design):** Successfully bridged the gap between Sellers and Buyers by designing unified user pools and separate RBAC roles.
- **Phase 6 (Security):** Implemented rigorous input sanitization and secure file upload methodologies (50MB S3 streaming instead of Buffer RAM bloat).
- **Phase 7 (Deployment):** Launched on AWS ECS.

## Lessons Learned
- PRISM's Phase 9 Testing phase caught several critical race conditions during checkout flows before deployment.
